# my AI (GenAI Chat App)

## 概要・説明
**my AI** は、複数の生成AIプロバイダー（OpenAI GPT、Anthropic Claude、Google Gemini、xAI Grok）を1つの洗練されたインターフェースで利用できる、ブラウザ完結型の高機能チャットアプリケーションです。

Obsidianを彷彿とさせるダークテーマ（Obsidian Terminal）を採用し、開発者やクリエイターが集中して作業できるUIを提供します。また、チャット履歴やAPIキーなどのデータはすべてブラウザのローカルデータベース（IndexedDB）に保存されるため、プライバシーが完全に保護されます。サーバーレスで動作するため、HTMLファイルをブラウザで開くだけですぐに利用を開始できます。

## 主な機能一覧
- **マルチAIプロバイダー対応**
  - GPT、Claude、Gemini、Grokをシームレスに切り替えて利用可能。
- **カスタムモデルと役割（システムプロンプト）の管理**
  - プロバイダーごとに独自のモデル名を追加可能。
  - 「プログラマー」「マーケター」などのプリセットの役割に加え、カスタムプロンプトによる自分だけの役割を作成・保存可能。
- **リッチなファイル添付機能**
  - 画像（プレビューおよび拡大表示対応）、テキスト、Markdownファイルの読み込みに対応。マルチモーダルなAIとの対話を実現。
- **高度なチャット履歴管理**
  - サイドバーでの履歴一覧表示、キーワード検索、チャットセッションの削除。
  - 指定したチャットのクリア機能。
- **ネイティブなファイルシステム連携（自動保存機能）**
  - File System Access APIを活用し、ローカルPCの指定フォルダにチャット履歴を直接・自動的に保存（JSON / Markdown / Text 形式から選択可能）。
- **インポート / エクスポート機能**
  - 手動での設定データおよびチャット履歴のバックアップと復元が可能。
- **快適なUX/UIとパフォーマンス**
  - メッセージ内のMarkdownレンダリング、コードハイライト、1クリックでのコードコピー機能。
  - Web Workerを用いたバックグラウンドでのAPI通信処理により、長文生成中もUIのフリーズを防止。
  - Screen Wake Lock APIによる、AI生成中の意図しない画面スリープを防止。
  - モバイル端末にも対応したレスポンシブデザイン。

## 使用技術・ライブラリ
本プロジェクトは、モダンなWeb標準APIを活用し、軽量かつ高速に動作するように設計されています。

- **コア技術**: HTML5, CSS3, Vanilla JavaScript (フレームワーク非依存)
- **データベース**: [Dexie.js](https://dexie.org/) (IndexedDBを扱いやすくするラッパーライブラリ)
- **Markdown処理**: [Marked.js](https://marked.js.org/) (チャットテキストのマークダウン変換)
- **セキュリティ**: [DOMPurify](https://github.com/cure53/DOMPurify) (XSS攻撃を防ぐためのHTMLサニタイズ)
- **UIコンポーネント**: Font Awesome 6.4.0 (アイコン), Google Fonts (Syne, DM Sans, JetBrains Mono)
- **主要なWeb API**:
  - `Web Workers API`: メインスレッドをブロックしない非同期ストリーミング処理
  - `File System Access API`: ローカルファイルシステムへの直接読み書き
  - `Screen Wake Lock API`: スリープ防止

## セットアップ・使い方

### 1. アプリの起動
環境構築やサーバーの立ち上げは一切不要です。
HTMLファイルをダウンロードし、**Google Chrome** や **Microsoft Edge** などのモダンブラウザにドラッグ＆ドロップして開くだけで起動します。
*(※ File System Access APIを活用したローカル自動保存機能を利用するため、Chrome/Edge系のブラウザを推奨します)*

### 2. 初期設定（APIキーの登録）
1. 画面右上の **歯車アイコン（設定）** をクリックして設定画面を開きます。
2. 「APIキー設定」セクションで、利用したいAIのAPIキー（Gemini, Claude, GPT, Grok）を入力します。
3. **「APIキーを保存」** ボタンをクリックします。
*(※ 入力したAPIキーはローカルブラウザ内にのみ保存され、外部サーバーに送信されることはありません)*

### 3. AIモデルと役割の選択
1. 画面下部のメッセージ入力欄の左側にある **ペンアイコン（AI設定）** をクリックします。
2. 対話する「役割」、「プロバイダー」、「モデル」を選択し、モーダル外をクリックして閉じます。
3. 新しいカスタムモデルや独自の役割を追加したい場合は、再度右上の設定画面から簡単に追加できます。

### 4. チャットの実行とファイル添付
- 入力欄にメッセージを入力し、**送信ボタン（紙飛行機アイコン）** をクリックしてチャットを開始します。
- メッセージ入力欄左側の **クリップボードアイコン** をクリックすると、画像やテキストファイルを添付してAIに解析させることができます。

### 5. ローカルへの自動保存の設定（任意・推奨）
設定画面の「ファイルシステム連携 (自動保存)」セクションにて以下を設定すると、チャットごとに自動でローカルPCへバックアップが作成されます。
1. 「保存形式」をMarkdownなどに設定します。
2. **「保存フォルダを選択」** をクリックし、PC内の安全なフォルダへのアクセスを許可します。
3. 以降、メッセージのやり取りが発生するたびに、選択したフォルダへ自動的にチャットログが書き出されます。
