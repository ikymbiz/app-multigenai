# multi AI

## 概要・説明
**multi AI** は、1つのインターフェースから複数の生成AIプロバイダー（OpenAI GPT、Anthropic Claude、Google Gemini、xAI Grok）を切り替えて利用できる、高機能なシングルページ・チャットアプリケーションです。

サーバーや複雑な環境構築は一切不要で、単一のHTMLファイルをブラウザで開くだけで動作します。洗練されたダークテーマ（Obsidian Terminal Design）を採用し、ブラウザのローカルストレージ（IndexedDB）やネイティブのファイルシステムと連携した高度な履歴管理・自動保存機能を備えています。

## 主な機能一覧

* **マルチLLMプロバイダー対応**
  * Gemini、Claude、GPT、GrokのAPIキーを設定画面から登録し、チャットごとにシームレスに切り替えて使用可能。
* **カスタムモデル・役割（ロール）管理**
  * 各プロバイダーの標準モデルに加え、独自のモデル名（例: `gpt-4-0613`など）を自由に追加・管理できます。
  * システムプロンプトを含む「役割」のプリセット（プログラマー、マーケター等）を搭載。カスタムロールの作成も可能です。
* **ストリーミング応答とバックグラウンド処理**
  * Web Workerを使用してAPI通信を行うため、AIのタイピング風ストリーミング応答中もブラウザの動作が重くなりません。
* **強力なローカル履歴管理**
  * すべてのチャット履歴・設定・ロールデータはブラウザ内（IndexedDB）に安全に保存されます。履歴の検索機能も完備。
  * **データの手動エクスポート/インポート**: 全データのJSONバックアップと復元が可能です。
* **ローカルファイルへの自動保存 (File System Access API)**
  * PC内の任意のフォルダを許可することで、チャット完了時にMarkdown、JSON、テキスト形式で会話履歴を自動でファイル保存します。
* **優れたUI/UX**
  * モバイルフレンドリーなレスポンシブデザイン。
  * AIの回答を美しく表示するMarkdownレンダリングとシンタックスハイライト（ワンクリックコピー対応）。
  * 誤ってページをリロードしても安心な入力中テキストの一時保存（レジューム機能）。
  * AI応答中の画面スリープを防止する Screen Wake Lock API 対応。

## 使用技術・ライブラリ

本プロジェクトはビルドツールを使用せず、ピュアなWeb技術と軽量なCDNライブラリで構成されています。

* **コア技術**: HTML5, CSS3, Vanilla JavaScript
* **データベース**: IndexedDB (ラッパーとして [Dexie.js](https://dexie.org/) を使用)
* **Markdown描画**: [Marked.js](https://marked.js.org/)
* **サニタイズ（XSS対策）**: [DOMPurify](https://github.com/cure53/DOMPurify)
* **アイコン**: FontAwesome 6
* **ブラウザAPI**: 
  * Web Workers API (非同期AI処理)
  * File System Access API (ローカルファイルへの直接保存)
  * Screen Wake Lock API (スリープ防止)

## セットアップ・使い方

### 1. アプリの起動
特別なインストールは必要ありません。ソースコードを `index.html`（または任意の名前）として保存し、Google Chrome や Microsoft Edge などのモダンブラウザで開くだけで起動します。
※ *File System Access API などの高度な機能を使用するため、Chromiumベースのブラウザを推奨します。*

### 2. APIキーの設定
1. 画面右上の **歯車アイコン（設定）** をクリックします。
2. 「APIキー設定」セクションで、利用したいプロバイダーのAPIキーを入力します。
3. **「APIキーを保存」** ボタンをクリックします。（キーはブラウザ内部にのみ暗号化されず保存されるため、共用PCでの利用にはご注意ください）

### 3. AIモデルと役割の選択
1. メッセージ入力欄の左側にある **ペンアイコン（AI設定）** をクリックします。
2. そのチャットで使用する「役割」「プロバイダー」「モデル」をドロップダウンから選択します。

### 4. チャットを行う
* 下部の入力エリアにメッセージを入力し、**紙飛行機アイコン** または `Enter` キーで送信します。（改行は `Shift + Enter` です）
* 左側のサイドバーから「新しいチャット」の作成や、過去の履歴の閲覧・検索が可能です（モバイルの場合は左上のメニューボタンからサイドバーを開きます）。

### 5. 自動保存機能の有効化（任意）
1. 設定画面を開き「ファイルシステム連携 (自動保存)」セクションに移動します。
2. **「保存フォルダを選択」** をクリックし、PC内のチャット履歴を保存したいフォルダを選択してブラウザに権限を許可します。
3. 保存形式（Markdown, JSON, テキスト）を選択すると、以降のチャットが自動的にそのフォルダへ出力されます。
