# my AI - GenAI Chat App

## 概要・説明
「my AI」は、複数の生成AIプロバイダー（OpenAI, Anthropic, Google, xAI）を一つのUIで切り替えて利用できる、高機能なローカルファースト・チャットアプリケーションです。

バックエンドサーバーを介さず、ブラウザから直接各プロバイダーのAPIと通信します。APIキーやチャット履歴、設定データはすべてブラウザのローカルストレージ（IndexedDB）に安全に保存されるため、プライバシーが保たれます。Obsidianを彷彿とさせる洗練されたダークテーマ（Obsidian Terminal風）を採用し、PC・スマートフォンの両方で快適に動作するレスポンシブデザインに対応しています。

## 主な機能一覧
- **マルチプロバイダー対応**: GPT (OpenAI), Claude (Anthropic), Gemini (Google), Grok (xAI) のAPIをサポート。
- **モデル・役割の柔軟なカスタマイズ**:
  - 各プロバイダーへのカスタムモデル（新モデルなど）の追加・編集機能。
  - システムプロンプト（役割）のプリセット選択、および新規プロンプトの追加・管理。
- **マルチモーダル対応（ファイル添付機能）**:
  - 画像ファイル（JPEG, PNG, GIF, WebP）の添付、サムネイルプレビュー、クリックでの拡大表示。
  - テキストファイル・Markdownファイルの読み込みとプロンプトへの結合。
- **完全ローカルファースト**:
  - Dexie.js (IndexedDB) を使用した高速かつ大容量な履歴のローカル保存。
  - これまでの全チャット履歴や設定の手動インポート / エクスポート（JSON形式）。
- **ファイルシステム連携 (自動保存)**:
  - File System Access APIを利用し、許可したローカルフォルダへチャット内容を直接・自動的に保存（JSON, Markdown, TXT形式から選択可能）。
- **リアルタイムストリーミング**:
  - Web Workerを活用し、メインスレッド（UI）をブロックすることなく、高速でなめらかなAI応答のストリーミング表示を実現。
- **リッチなテキストUI**:
  - Markdownのパース表示（リスト、テーブル、コードブロック等に対応）。
  - ワンクリックでAIの回答をコピーできる機能。
- **その他の便利な機能**:
  - サイドバーでの履歴検索とセッション管理（削除・新規作成）。
  - AI生成中の画面スリープを防ぐ Wake Lock API 連携。
  - 入力途中のメッセージを自動保持するレジューム機能。

## 使用技術・ライブラリ
- **フロントエンド**: HTML5, CSS3, Vanilla JavaScript (フレームワーク不使用の軽量設計)
- **データベース**: [Dexie.js](https://dexie.org/) (IndexedDBラッパー)
- **テキスト・Markdown処理**: 
  - [marked.js](https://marked.js.org/) (Markdownパーサー)
  - [DOMPurify](https://github.com/cure53/DOMPurify) (XSS対策 / HTMLサニタイザー)
- **UI / デザイン**: 
  - Font Awesome (v6.4.0)
  - Google Fonts (Syne, DM Sans, JetBrains Mono)
- **活用しているモダン Web APIs**: 
  - Web Workers API (非同期ストリーミング処理)
  - File System Access API (ローカルディレクトリへのファイル保存)
  - Screen Wake Lock API (画面スリープ防止)

## セットアップ・使い方

### 1. 起動方法
本アプリケーションはフロントエンドのみで完結しているため、複雑なビルド（npm install等）は不要です。ただし、Web WorkerやFile System Access APIなどのモダン機能を使用するため、**ローカルサーバー経由での起動**が必要です。

1. 本ソースコード（HTMLファイル）をダウンロード、またはリポジトリをクローンします。
2. ローカルサーバーを立ち上げます。
   - **VSCodeをご利用の場合**: 拡張機能「Live Server」をインストールし、HTMLファイルを右クリックして「Open with Live Server」を選択します。
   - **Pythonをご利用の場合**: ターミナルでファイルのあるディレクトリに移動し、以下のコマンドを実行します。
     ```bash
     python -m http.server 8000
     ```
3. ブラウザを開き、`http://localhost:8000` にアクセスします。

### 2. 初期設定（APIキーの登録）
1. 画面右上の **歯車アイコン（設定）** をクリックして設定画面を開きます。
2. 「APIキー設定」セクションで、利用したいAIプロバイダーのAPIキーを入力します。
3. **「APIキーを保存」** ボタンをクリックします。
   *※ 入力したAPIキーはあなたのブラウザ内（IndexedDB）にのみ保存され、開発者や第三者に送信されることはありません。*

### 3. 基本的な使い方
- **チャットの開始**: アプリを起動すると「新しいチャット」が開きます。過去の履歴は左側のサイドバー（スマホの場合は左上のメニューボタンから展開）から確認・検索できます。
- **AIモデル・役割の変更**: 入力欄の左側にある **ペンアイコン（AI設定）** をクリックすると、そのチャットで使用する「役割（システムプロンプト）」「プロバイダー」「モデル」を変更できます。
- **ファイルの添付**: 入力欄の左側にある **クリップアイコン** をクリックして、画像やテキストファイルを添付できます（1ファイル最大5MB）。
- **メッセージの送信**: テキストエリアにメッセージを入力し、右下の送信ボタン（紙飛行機アイコン）を押すか、プロンプトを送信します。
- **チャットのローカル自動保存**: 設定画面の「ファイルシステム連携」から **「保存フォルダを選択」** をクリックし、PC内の任意のフォルダへのアクセスを許可すると、チャットが更新されるたびに指定フォルダへ自動でファイルが書き出されます。
