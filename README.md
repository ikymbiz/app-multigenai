# multi AI

## 概要

**multi AI** は、1つの洗練されたインターフェースから複数の生成AI（Gemini, Claude, ChatGPT, Grok）をシームレスに利用できる、ブラウザ完結型のチャットアプリケーションです。

「Obsidian Terminal」風の美しくモダンなダークUIを採用し、快適なタイピング体験を提供します。バックエンドサーバーを必要とせず、APIキーはすべてブラウザのローカルデータベース（IndexedDB）に安全に保存されます。また、ブラウザから直接ローカルのPCフォルダへチャット履歴を自動保存する高度な機能も搭載しています。

## 主な機能

- **マルチAIプロバイダー対応**
  - Google Gemini, Anthropic Claude, OpenAI GPT, xAI (Grok) の主要4大プロバイダーをサポート。
  - チャットごとにプロバイダーとモデルを即座に切り替え可能。
- **カスタムモデルと役割（システムプロンプト）の管理**
  - 各プロバイダーに最新のカスタムモデルを自由に追加・編集可能。
  - 「プログラマー」「マーケター」など、任意のシステムプロンプトを持たせた「役割」を作成し、チャットの目的に合わせて適用できます。
- **高度なローカルデータ管理**
  - **IndexedDB**: すべてのチャット履歴と設定はブラウザ内に保存されます。
  - **ファイルシステム連携 (自動保存)**: File System Access APIを活用し、指定したローカルPCのフォルダにチャットのログを直接自動保存（JSON, Markdown, TXT形式）。
  - **インポート / エクスポート**: 環境移行のための全データの一括バックアップと復元機能。
- **快適なチャットUI / UX**
  - Markdown対応（シンタックスハイライト、コードのワンクリックコピー）。
  - Web Workerを利用した非同期処理による、スムーズなストリーミング応答（文字がリアルタイムに出現）。
  - モバイル対応のレスポンシブデザインとスライド式サイドバー。
  - 誤操作防止の自動レジューム（入力中テキストの保存）機能。

## 使用技術・ライブラリ

本プロジェクトは、ビルドツール不要の「シングルHTMLファイル（Vanilla JS）」として構築されています。

- **コア技術**: HTML5, CSS3, JavaScript (ES6+)
- **ブラウザAPI**: 
  - Web Workers API (ストリーミング通信のバックグラウンド処理)
  - File System Access API (ローカルファイルへの直接書き込み)
  - Screen Wake Lock API (生成中のスリープ防止)
- **外部ライブラリ**:
  - [Dexie.js](https://dexie.org/): IndexedDBの操作を簡略化するデータベースラッパー
  - [Marked.js](https://marked.js.org/): Markdownの高速レンダリング
  - [DOMPurify](https://github.com/cure53/DOMPurify): XSS攻撃を防ぐHTMLサニタイズ
  - [FontAwesome](https://fontawesome.com/): UIアイコン
- **フォント**: Syne, DM Sans, JetBrains Mono (Google Fonts)

## セットアップ・使い方

### 1. 起動方法
サーバーやNode.jsの環境構築は一切不要です。
ダウンロードしたHTMLファイルを、モダンブラウザ（Google Chrome, Microsoft Edge推奨）で開くだけで起動します。
*※ File System Access APIなどの高度な機能を使用するため、ローカルサーバー（VSCodeのLive Serverなど）経由での起動を推奨します。*

### 2. 初期設定（APIキーの登録）
1. 画面右上の **歯車アイコン（設定）** をクリックします。
2. 「APIキー設定」セクションで、利用したいAIのAPIキーを入力します。
3. **「APIキーを保存」** ボタンをクリックします。
   *※ キーはブラウザ内部にのみ保存され、外部サーバーに送信されることはありません。*

### 3. チャットの開始
1. メイン画面下部の入力欄の左にある **ペンアイコン（AI設定）** をクリックします。
2. 使用する「役割（プロンプト）」「プロバイダー」「モデル」を選択します。
3. メッセージを入力し、送信ボタン（またはEnterキー）を押すとAIとの対話が開始されます。

### 4. ローカルフォルダへの自動保存（オプション）
1. 設定画面の「ファイルシステム連携」セクションを開きます。
2. 保存形式（JSON, Markdown, テキスト）を選択します。
3. **「保存フォルダを選択」** をクリックし、PC内の任意のフォルダへのアクセスを許可します。
4. 以降、チャットを行うたびに、指定したフォルダに自動的にログファイルが生成・更新されます。
