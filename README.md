# multi AI

## 概要・説明
**multi AI** は、複数の主要な生成AIプロバイダー（OpenAI GPT、Anthropic Claude、Google Gemini、xAI Grok）を1つのインターフェースで統合的に利用できる、ローカルファーストなWebチャットアプリケーションです。

バックエンドサーバーを一切必要とせず、ブラウザ上で完結して動作します。すべてのAPI通信はクライアントサイドから直接行われ、チャット履歴や設定、APIキーなどのデータはすべてブラウザのローカルデータベース（IndexedDB）に安全に保存されます。

美しいダークテーマ（Obsidian風デザイン）を採用し、マルチデバイスに対応。Web Workerを活用した非同期のストリーミング描画により、快適でスムーズなAI対話体験を提供します。

## 主な機能一覧

* **🤖 マルチプロバイダー対応**
  * Gemini, Claude, GPT, Grok のAPIをサポートし、設定画面からシームレスに切り替え可能。
* **⚙️ モデルとシステムプロンプトのカスタマイズ**
  * 各プロバイダーに独自の新しいモデル名を手動で追加可能。
  * 「プログラマー」「マーケター」などのプリセット役割のほか、カスタムのシステムプロンプト（役割）を自由に作成・保存できます。
* **💾 ローカルファーストのデータ管理**
  * チャット履歴や設定はすべてローカル（IndexedDB）に保存されるため、プライバシーが保護されます。
  * JSON形式での手動エクスポート / インポートによるデータバックアップ機能も搭載。
* **📁 ファイルシステム自動連携**
  * File System Access API を使用し、指定したローカルフォルダへチャット内容を直接・自動的に保存可能（Markdown / JSON / Text 形式に対応）。
* **⚡ 高速でブロックのないUI**
  * AIのレスポンス処理に Web Worker を採用。重い処理でも画面がフリーズせず、リアルタイムなストリーミング表示を実現します。
* **📱 レスポンシブ & リッチなUI**
  * スマートフォン、タブレット、PCに対応したモダンなデザイン。
  * Markdownレンダリング、シンタックスハイライト、ワンクリックでのコードコピー機能を搭載。
  * メッセージ入力中の自動バックアップ（レジューム機能）対応。

## 使用技術・ライブラリ

### コア技術
* **HTML5 / CSS3 / Vanilla JavaScript**: フレームワークレスで軽量・高速に動作。
* **Web APIs**:
  * **IndexedDB**: ローカルデータ保存。
  * **Web Workers API**: バックグラウンドでのAPI通信とストリーミング処理。
  * **File System Access API**: ローカルディレクトリへのファイル直接書き込み。
  * **Screen Wake Lock API**: 生成中の画面スリープ防止。

### 外部ライブラリ (CDN経由)
* **[Dexie.js](https://dexie.org/)**: IndexedDBを簡単に扱うためのラッパーライブラリ。
* **[Marked.js](https://marked.js.org/)**: AIからのマークダウン応答をHTMLに変換。
* **[DOMPurify](https://github.com/cure53/DOMPurify)**: XSS攻撃を防ぐためのHTMLサニタイズ。
* **[Font Awesome](https://fontawesome.com/)**: UIアイコンセット。

## セットアップ・使い方

### 1. 起動方法
サーバーの構築や依存パッケージのインストールは不要です。
提供されたHTMLファイル（例: `index.html`）をダブルクリックし、モダンブラウザ（Google Chrome, Microsoft Edge など）で開くだけで起動します。

### 2. 初期設定（APIキーの登録）
AIとチャットを行うためには、各プロバイダーのAPIキーが必要です。
1. 画面右上の **歯車アイコン（設定）** をクリックします。
2. 「APIキー設定」セクションで、利用したいAIのAPIキーを入力します。
3. **「APIキーを保存」** ボタンをクリックします。

### 3. チャットの開始
1. 左サイドバー（スマホの場合は左上のメニューボタンから展開）の **「+ 新しいチャット」** をクリックします。
2. メッセージ入力欄の左側にある **ペンアイコン（AI設定）** をクリックし、使用する「役割」「プロバイダー」「モデル」を選択します。
3. メッセージを入力し、右下の送信ボタン（またはEnterキー）を押すと、AIからの回答がリアルタイムで表示されます。

### 4. 高度な使い方
* **ファイルへの自動保存**
  * 設定画面の「ファイルシステム連携」から **「保存フォルダを選択」** をクリックし、PC内のフォルダを指定してブラウザに権限を与えます。以降、チャットが完了するたびに、そのフォルダにファイルが自動保存されます。
* **カスタムモデルの追加**
  * 各APIプロバイダーから新しいモデルが発表された場合、設定の「AIモデル管理」からプロバイダーを選び、手動でモデル名（例: `gpt-4o-mini`）を入力して追加できます。
* **データの引き継ぎ**
  * 別のブラウザや端末に移行する場合は、設定の「データ管理」から **エクスポート** を行い、移行先で **インポート** を実行してください。
