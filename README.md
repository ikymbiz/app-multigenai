# my AI (GenAI Chat App)

## 概要

**my AI** は、1つのインターフェースから複数の主要な生成AI（LLM）と対話できる、ブラウザ完結型の高機能チャットアプリケーションです。

OpenAI (GPT)、Anthropic (Claude)、Google (Gemini)、xAI (Grok) の各APIキーを設定するだけで、バックエンドサーバーを構築することなくブラウザ上から直接AIと通信します。データはすべてローカル（ブラウザのIndexedDB）に保存されるため、プライバシーとセキュリティを確保しながら快適に利用できます。

ダークモードを基調とした洗練された「Obsidian Terminal」デザインシステムを採用し、モバイル端末にも対応したレスポンシブなUIを提供します。

## 主な機能

* **マルチLLM対応 & カスタムモデル設定**
  * ChatGPT, Claude, Gemini, Grokの主要プロバイダーを統合。
  * デフォルトのモデルに加え、ユーザー自身で新しいモデル名を追加可能（例: 最新モデルがリリースされた際も即座に対応可能）。
* **役割（システムプロンプト）の管理**
  * 「アシスタント」「プログラマー」「マーケター」などのプリセットロールを用意。
  * 独自のシステムプロンプトを持つカスタムロールを自由に作成・編集・削除可能。
* **マルチモーダル入力（ファイル添付）**
  * 画像ファイル（JPEG, PNG, GIF, WebP）の添付とAIによる画像認識をサポート。
  * テキストファイル（TXT, Markdown）を読み込み、コンテキストとしてプロンプトに含めることが可能。
* **高度なチャット履歴・メッセージ管理**
  * チャット履歴は自動的にローカルデータベース（IndexedDB）に保存。
  * 過去のメッセージの編集、特定のユーザーメッセージからの「再生成（Redo）」、不要なメッセージの削除が可能。
  * AIの返答をMarkdownで美しくレンダリングし、ワンクリックでテキストコピー可能。
  * ストリーミングレスポンス（Web Workerを利用）により、AIの返答をリアルタイムに表示。
* **ファイルシステム連携 & データバックアップ**
  * File System Access APIを活用し、ローカルPCの指定フォルダへチャット履歴を自動保存（JSON, MD, TXT形式を選択可能）。
  * アプリ全体の設定や履歴データの手動エクスポート・インポート機能。

## 使用技術・ライブラリ

本プロジェクトは、モダンなWeb標準技術を活用し、ビルドツール不要のシングルファイル（HTML/CSS/JS）で構成されています。

* **フロントエンド**: HTML5, CSS3, Vanilla JavaScript
* **非同期処理**: Web Workers (UIをブロックしないAPIストリーミング通信)
* **ローカルDB**: [Dexie.js](https://dexie.org/) (IndexedDBラッパー)
* **マークダウン処理**:
  * [Marked.js](https://marked.js.org/) (Markdownパーサー)
  * [DOMPurify](https://github.com/cure53/DOMPurify) (XSS対策サニタイザ)
* **アセット・UI**:
  * Font Awesome 6 (アイコン)
  * Google Fonts (Syne, DM Sans, JetBrains Mono)
* **ブラウザAPI**: File System Access API, Wake Lock API

## セットアップ・使い方

本アプリケーションはバックエンドサーバーを必要としません。HTMLファイルをブラウザで開くだけですぐに使用を開始できます。

### 1. 起動方法
1. 本リポジトリの `index.html`（または提供されたHTMLファイル）をダウンロードします。
2. ダウンロードしたファイルを Google Chrome, Microsoft Edge, Safari などのモダンブラウザで開きます。

### 2. 初期設定（APIキーの登録）
AIと対話するためには、各プロバイダーのAPIキーが必要です。
1. 画面右上の **歯車アイコン（設定）** をクリックします。
2. 「APIキー設定」セクションで、利用したいAIのAPIキーを入力します。
3. **「APIキーを保存」** ボタンをクリックします。
   *(※ APIキーはブラウザのローカルデータベースにのみ保存され、外部サーバーに送信されることはありません)*

### 3. チャットの開始
1. メッセージ入力欄の左側にある **ペンアイコン（AI設定）** をクリックします。
2. 利用したい「プロバイダー」「モデル」「役割（システムプロンプト）」を選択します。
3. メッセージを入力し、**送信ボタン** または `Enter` キー（複数行の場合は `Shift + Enter` など）を押して対話を開始します。
4. クリップのアイコンをクリックすると、画像やテキストファイルを添付できます。

### 4. 高度な使い方
* **メッセージの編集・再生成**: メッセージをクリック（またはタップ）すると、オーバーレイメニューが表示されます。そこから編集（鉛筆マーク）や再生成（リロードマーク）、削除（ゴミ箱マーク）が行えます。
* **自動保存設定**: 設定画面の「ファイルシステム連携」からローカルPCの保存フォルダを選択すると、チャットの会話内容が自動的に指定したフォーマット（MD等）でファイルとして保存されます。

---

**⚠️ 注意事項**
* 本アプリケーションは直接各プロバイダーのAPIエンドポイントと通信します。そのため、ブラウザのCORSポリシーの制限を受けない環境、または公式で直接ブラウザからのアクセスが許可されているエンドポイントを利用しています。
* Claude APIを使用する際など、一部の環境では `anthropic-dangerous-direct-browser-access` ヘッダーを用いてブラウザからの直接アクセスを許可する仕様になっています。APIキーの取り扱い（第三者への漏洩など）には十分ご注意ください。
