# multi AI

## 概要・説明
「multi AI」は、複数の主要な生成AI（OpenAI GPT、Anthropic Claude、Google Gemini、xAI Grok）を1つのインターフェースから利用できる、ブラウザ完結型の高機能チャットアプリケーションです。

バックエンドサーバーを必要とせず、ブラウザから直接各プロバイダーのAPIに通信を行うため、HTMLファイルをブラウザで開くだけですぐに利用を開始できます。また、チャット履歴やAPIキーなどの設定情報はすべてブラウザ内のローカルデータベース（IndexedDB）に保存されるため、セキュアかつプライバシーが保たれた環境でAIを活用できます。

## 主な機能一覧

* **マルチプロバイダー対応**: 1つのアプリ内で GPT、Claude、Gemini、Grok をシームレスに切り替えて利用可能。
* **モデルの柔軟な管理**: 各プロバイダーの最新モデル（デフォルト設定）に加え、ユーザー自身でカスタムモデルを追加・編集・削除可能。
* **システムプロンプト（役割）管理**: アシスタントやプログラマーなどのプリセット役割のほか、独自のシステムプロンプトを設定したオリジナルの役割を作成可能。
* **ローカル履歴管理**: チャット履歴は自動的にローカル保存され、サイドバーから過去の会話を検索・再開・削除が可能。
* **リアルタイムストリーミング**: Web Workerを活用した非同期ストリーミング処理により、UIをフリーズさせることなくAIの応答をリアルタイムに表示。
* **Markdown & コードハイライト**: AIの応答をMarkdownとして美しくレンダリングし、コードブロックのワンクリックコピー機能も搭載。
* **ローカルフォルダへの自動保存**: File System Access APIを利用し、指定したローカルPCのフォルダへチャット内容を自動でバックアップ（JSON / Markdown / テキスト形式から選択可能）。
* **データのエクスポート / インポート**: 全ての設定とチャット履歴をJSONファイルとして書き出し・読み込み可能。
* **モバイル最適化**: スマートフォンやタブレットでも快適に操作できるレスポンシブデザイン。
* **便利な拡張機能**: 
  * 回答生成中の画面スリープを防止（Screen Wake Lock API）
  * 入力途中のテキストを自動保存し、リロード時にも復元（localStorage）
  * 現在のチャット画面のワンクリッククリア機能

## 使用技術・ライブラリ

ビルドツールやフレームワークを使用しない、軽量な Vanilla JavaScript 構成です。

* **コア技術**: HTML5, CSS3, Vanilla JavaScript
* **データベース**: [Dexie.js](https://dexie.org/) (IndexedDBのラッパーライブラリ)
* **Markdown描画**: [Marked.js](https://marked.js.org/)
* **サニタイズ (XSS対策)**: [DOMPurify](https://github.com/cure53/DOMPurify)
* **UIアセット**: Font Awesome 6.4.0 (アイコン), Google Fonts (Syne, DM Sans, JetBrains Mono)
* **ブラウザAPI**: Web Workers, File System Access API, Screen Wake Lock API

## セットアップ・使い方

### 1. 起動方法
本アプリケーションは完全にクライアントサイドで動作しますが、ブラウザのセキュリティ制限（Web WorkerやFile System Access APIの利用制限）を回避するため、ローカルサーバー経由での起動を推奨します。

1. 本リポジトリのHTMLファイルをダウンロードします。
2. VSCodeの拡張機能「Live Server」や、Pythonのローカルサーバー（`python -m http.server`）などを使用してHTMLファイルをブラウザで開きます。

### 2. 初期設定（APIキーの登録）
1. 画面右上の **歯車アイコン（設定）** をクリックします。
2. 「APIキー設定」セクションにて、利用したいAIプロバイダーのAPIキーを入力し、「APIキーを保存」ボタンをクリックします。
   * *※APIキーはブラウザのローカルデータベースに暗号化されずに保存されます。共有PCでの利用にはご注意ください。*

### 3. チャットの開始
1. 画面左下（入力欄の左側）にある **鉛筆アイコン（AI設定）** をクリックします。
2. 使用したい「役割（システムプロンプト）」、「プロバイダー」、「モデル」を選択します。
3. 入力欄にメッセージを入力し、Enterキー（または送信ボタン）でAIに送信します。

### 4. 便利な設定
* **役割 / モデルの追加**: 設定画面から、用途に合わせた独自のシステムプロンプトや、プロバイダーの新しいモデル名を追加できます。
* **ファイルシステム連携**: 設定画面の「ファイルシステム連携 (自動保存)」から保存先フォルダを選択すると、チャットのたびに指定した形式（md, txt, json）でローカルPC内に会話ログが自動保存されます。
* **バックアップと移行**: 設定画面下部の「データ管理」から、現在のすべてのデータをエクスポートし、別のブラウザや端末にインポートして環境を移行できます。
